About this Sudoku Solver

This is a simple text-based Sudoku solver, written in Python 2.7.6. This project was developed for the Insight Data Engineering Fellows Program application for the January 2015 session.
This also presented a great opportunity to learn a new technology. After first implementing the Sudoku Solver in Python, I went on create a web app out of it using the Django framework. I must say, it was a fun learning experience.
